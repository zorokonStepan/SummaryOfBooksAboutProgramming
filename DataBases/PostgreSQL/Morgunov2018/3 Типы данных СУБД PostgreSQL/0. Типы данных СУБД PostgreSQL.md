### 1. Числовые типы
```
    1.1. целочисленные типы
        1.1.1. smallint
        1.1.2. integer
        1.1.3. bigint
        
    1.2. числа фиксированной точности
        1.2.1. numeric
        1.2.2. decimal
    
    1.3. типы данных с плавающей точкой
        1.3.1. real              # от 1E−37 до 1E+37 с точностью не меньше 6 десятичных цифр
        1.3.2. double precision  # от 1E−307 до 1E+308 с точностью не меньше 15 десятичных цифр
        1.3.3. float
    
    1.4. последовательные типы (serial)        
        1.4.1. smallserial
        1.4.2. serial
        1.4.3. bigserial
```

### 2. Символьные (строковые) типы
```
    2.1. character varying(n) OR varchar(n)
    2.2. character(n) OR char(n)
    2.3. text
```

### 3. Типы «дата/время»
```
    1. date                 # default:«yyyy-mm-dd»
    2. time                 # хранит время суток
    3. time with time zone  # хранит время суток + часовой пояс
    4. timestamp            # временная отметка = дата + время
    5. timestamp with time zone(timestamptz)
    6. interval  # продолжительность отрезка времени между двумя моментами времени
```
