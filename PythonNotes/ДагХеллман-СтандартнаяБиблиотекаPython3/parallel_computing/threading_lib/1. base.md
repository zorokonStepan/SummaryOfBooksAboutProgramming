### Объекты Thread
```
    def worker():
        print('Worker')    
    
    for _ in range(5):
        threading.Thread(target=worker).start()
```

### Передача аргументов
```
    def worker(num):
        print('Worker: %s' % num)    
    
    for i in range(5):
        threading.Thread(target=worker, args=(i,)).start()    
```
